## 进制转换
- 十进制 -> 二进制：除2取余
- 二进制 -> 十进制：8421码
- 二进制数系统中，每个0或1就是一个位，叫做bit（比特）

## 字节
字节是计算机中最小的存储单元。计算机中存储的任何数据，都是以字节的形式存储的。

- 1TB = 1024GB
- 1GB = 1024MB
- 1MB = 1024KB
- 1KB = 1024B
- 1B = 1024bit
